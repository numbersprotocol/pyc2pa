## Injecting into an image

The CAI-Tool currently does not automatically inject the CAI hex code into an JPG. You will need to use third party software. I suggest using [010 Editor](https://www.sweetscape.com/010editor/).

### 010 Editor Steps

1) Open Image in 010 Editor
2) Edit -> Insert/Overwrite -> Insert Bytes
3) Input size output from CAI-Tool script
4) Copy hex output from CAI-Tool script
5) Edit -> Paste From -> Hex to Text


## Injecting XMP 

1) Utilize `exiv2`
2) Run `exiv2 -M'set Xmp.dcterms.provenance self#jumbf=cai/<store label>/cai.claim' <JPG Filename>`
3) Confirm run `exiv2 -p a pr <JPG Filename>`
